# El Hijo Pródigo

## Requisitos

- Python 3x
- pip

## Instalación

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Ejecución

```bash
venv/bin/python3 ejemplo.py
```
